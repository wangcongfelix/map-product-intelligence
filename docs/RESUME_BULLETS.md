# Resume Bullets

## 一句话项目名

地图产品竞品分析 Agent｜AI 用户声音与竞品信号工作台

## 中文简历版

- 设计并实现面向地图与户外产品经理的 AI 用户声音分析工作台，覆盖资料导入、AI 预分析、人工审核、信号沉淀、图表分析和报告输出的完整闭环。
- 基于 DeepSeek 和结构化 Schema 搭建产品信号预分析流程，将应用评论、版本日志、社区反馈和竞品体验记录转化为可审核的产品模块、信号类型、影响程度和建议动作。
- 设计 Human-in-the-loop 审核机制，支持产品经理对 AI 结果进行通过、修改或忽略，确保进入信号总表和报告中心的数据均经过人工确认。
- 构建报告中心，基于已确认信号生成竞品与用户声音分析图表、PNG 图表导出和周报/月报风格 Markdown 报告。
- 保留 Mock AI 兜底和回归测试脚本，保障无 API Key 场景下仍可完整演示产品流程，并提升 AI 分析链路的可测试性。

## English Resume Version

- Designed and built an AI-powered product intelligence workspace for map and outdoor product managers, covering material import, AI pre-analysis, human review, signal management, charts, and PM report generation.
- Integrated DeepSeek with a structured analysis schema to transform app reviews, release notes, community feedback, and competitor observations into reviewable product signals.
- Designed a human-in-the-loop workflow where PMs can approve, modify, or ignore AI outputs, ensuring that only human-confirmed signals enter the signal table and report center.
- Built a report center that generates competitor and user-voice charts, PNG chart exports, and weekly/monthly-style Markdown reports from confirmed signals.
- Added Mock AI fallback and regression scripts to support demos without API keys and improve testability of the AI analysis workflow.
